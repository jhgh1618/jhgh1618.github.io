#  1. 정부24 사이트에 접속
아래의 링크를 클릭하면 정부24 사이트로 이동합니다.<br><br>
[정부24사이트 링크](https://www.gov.kr/portal/main/nologin)

# 2. 메인 화면에서 주민등록 아이콘 클릭
주민등록등본 발급 아이콘을 눌러 주세요.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img1.png" width="100%" height="100%" title="주민등록아이콘"/><br><br>

# 3. 민원서비스 페이지에서 발급하기를 클릭
주민등록표 등본(초본) 발급 맨 밑을 보시면 발급하기 버튼이 있습니다. 버튼을 눌러 주세요.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img2.png" width="100%" height="100%" title="발급버튼"/><br><br>

# 5. 로그인 또는 비회원신청을 선택
회원 또는 비회원으로 신청 가능합니다. 저는 비회원으로 진행해 보겠습니다. 회원이신 분은 회원을 클릭하여 로그인 하시면 됩니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img3.png" width="100%" height="100%" title="로그인버튼"/><br><br>

개인정보에 동의해 줍니다. 이것을 클릭안하면 다음 진행이 안됩니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img4.png" width="100%" height="100%" title="동의"/><br><br>

빨간색 3칸에 개인정보를 넣고 확인 버튼을 눌러줍니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img5.png" width="100%" height="100%" title="개인정보확인"/><br><br>

# 6. 등본 또는 초본 신청 내용을 선택합니다.
펼치기 버튼을 눌러 발급할 문서를 선택합니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img6.png" width="100%" height="100%" title="개인정보확인"/><br><br>

주소를 선택하고 발급형태를 선택합니다. 선택발급을 선택하면 주민등록번호 뒷자리 공개 여부 등을 선택할 수 있습니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img7.png" width="100%" height="100%" title="개인정보확인"/><br><br>

저는 프린터로 종이 문서를 뽑을 것이라서 온라인발급(본인출력)을 선택합니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img8.png" width="100%" height="100%" title="개인정보확인"/><br><br>

# 7. 문서 출력을 위해 인증 진행합니다.
인증을 위해 여러 방법이 있는 데, 요즘은 네이버나 카카오톡 ID는 누구나 있어서 간편인증으로 진행합니다. 간편인증 클릭해주세요.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img9.png" width="100%" height="100%" title="개인정보확인"/><br><br>

네이버나 카카오를 선택하고 개인정보 입력, 동의 선택해주세요.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img9-1.png" width="100%" height="100%" title="개인정보확인"/><br><br>

인증 요청 클릭하면 아래 화면이 뜨는 데, 휴대폰으로 인증 진행 완료하고 인증 완료 버튼을 눌러주세요.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img9-2.png" width="100%" height="100%" title="개인정보확인"/><br><br>

# 8. 문서를 출력합니다.
서비스 신청내역에서 문서출력을 클릭합니다. <br>
<img src="/assets/images/RegistrationCertificate_20240614/img10.png" width="100%" height="100%" title="개인정보확인"/><br><br>

새로운 창이 뜨는데 오른쪽 위부분을 보면 인쇄 버튼이 있습니다. 클릭하시면 인쇄 진행할 수 있습니다.<br>
<img src="/assets/images/RegistrationCertificate_20240614/img11.png" width="100%" height="100%" title="개인정보확인"/><br><br>
